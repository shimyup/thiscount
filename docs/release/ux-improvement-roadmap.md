## 🛠 Phase 2 — 다음 세션 우선순위 (MAU 3K 돌파 전)

### 1. 공유 카드 자동 생성 ⭐ 최우선
**목표**: 받은 편지를 인스타 스토리에 바로 공유 (바이럴 루프 핵심)

**구현 사양**:
- 1080x1920 이미지
- 발신국 국기 + 지도 위 여정 썸네일 + 편지 내용 1~2줄
- Thiscount 로고 워터마크
- Flutter Canvas 기반 (오프라인 생성, 비용 0)

**예상 공수**: 2~3일  
**파일**: `lib/features/share/share_card_service.dart` 신규

### 2. 홈 화면 단순화 (지도 + FAB만)
**목표**: 1 Screen = 1 Purpose 원칙, 인지 부담 ↓

**Before → After**:
- 상단 검색바·뱃지·카운터 제거
- 하단 "편지 쓰기" FAB만 거대하게
- 타워 세부 통계는 Tower 탭으로 이동
- 실시간 카운터(Live 1234명)는 온보딩 슬라이드 1회만

**예상 공수**: 1~2일  
**파일**: `lib/features/map/screens/world_map_screen.dart` 리팩토링

### 3. 온보딩 점진 공개 (Progressive Disclosure)
**목표**: 신규 유저 첫 3일간 복잡 기능 감추기

**단계별 해금**:
- Day 0: 기본 편지 작성 + 지도 탐색만
- Day 1-3: 첫 답장 → Tower 레벨 공개
- Day 4-7: 10통 달성 → "오늘의 편지" 해금
- Day 8-14: 편지지·폰트 커스텀 해금
- Day 15+: 주변 줍기 + Premium 소개

**예상 공수**: 2~3일  
**파일**: `lib/state/app_state.dart` UserProgress 신규 + 각 feature gating

### 4. 일일 스트릭 + 주간 챌린지
**목표**: D7 리텐션 30% 목표 달성

**구현**:
- 일일 스트릭 카운터 (마지막 접속일 기반)
- 주간 챌린지: "이번 주 5개 대륙에 편지 보내기" → 보상: 주말 한정 테마
- 홈 화면 상단 작은 뱃지

**예상 공수**: 3~5일

## 🌟 Phase 3 — 매력도 극대화 (MAU 10K 돌파 전)

### 5. "오늘의 편지" 매일 오전 8시 푸시
- FCM 푸시 알림 연동 전제
- 메시지 예: "아침 편지가 도착했어요 ☕️ 🇧🇷 브라질 리우에서"
- 예상 개봉률 30~45%

### 6. 이번 달의 도시 큐레이션
- 매달 도시 1개 큐레이션
- 해당 도시로 편지 보낼 때 자동 테마 적용 (벚꽃 · 오로라 · 향신료 등)
- 관광청·문화원 제휴 가능

### 7. 편지 개봉 애니메이션 업그레이드
- 봉투가 화면 아래에서 떠오름
- 종이 접힘 풀리는 햅틱 (iOS 3회, Android 진동)
- 타자기 스타일 텍스트 출현 (첫 3줄)

### 8. "희소성" 표현
- Letter.maxReaders=3 기능을 UI에 노출
- "이 편지를 받은 사람은 당신 외에 2명"
- "곧 다른 바다로 떠납니다" 카운트다운

## 🚫 하지 말 것 (Anti-patterns)

### Slowly 따라하기
- Slowly의 직접 답장 모델, 스탬프 수집 등은 이미 다 있음
- **차별점**: Thiscount는 "지도 위 나의 타워" + "실시간 배송 궤적" + "주변 줍기"

### 알고리즘 소셜
- "추천 편지", "피드 정렬", "인기 편지" 등 타 소셜 용어 금지
- 무작위성 + 우연성 + 배송 시간이 핵심

### 과도한 게이미피케이션
- 포인트, 배지, 레벨은 최소화
- 탑 레벨은 "집이 크다"는 시각 표현 정도로만

### Description (Korean)
Thiscount는 전 세계 사용자와 편지를 주고받는 지도 기반 소셜 앱입니다.

편지를 보내면 지도 위에서 이동 경로가 표시되고, 도착까지의 기다림이 새로운 설렘이 됩니다.
가벼운 한 줄 인사부터 진심을 담은 긴 편지까지, 익명 기반으로 부담 없이 시작할 수 있습니다.

주요 기능
- 세계 지도 기반 편지 배송 여정
- 국가 랜덤 발송 및 도착 편지함
- 답장 중심의 느린 소통 경험
- 프리미엄: 하루 최대 30통, 이미지+링크 편지, 특급 배송(하루 3통)
- Brand/Creator: 하루 최대 200통, 인증 배지, 대량 발송 및 추가 발송권 구매

추천 대상
- 빠른 채팅보다 의미 있는 소통을 선호하는 사용자
- 다른 나라 사람들과 부담 없이 연결되고 싶은 사용자
- 감성 기반 커뮤니케이션과 글로벌 탐험을 함께 즐기고 싶은 사용자

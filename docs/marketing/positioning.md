# LetterGo Positioning

## Product One-liner
LetterGo는 전 세계 낯선 사람과 감정 중심 편지를 주고받고, 지도 위 배송 여정을 통해 관계를 쌓는 감성 소셜 앱입니다.

## Target User (Primary)
- 18-34세, 텍스트 기반 소셜/감성 콘텐츠 선호 사용자
- 인스타/틱톡 피로도가 높고, 느린 소통을 원하는 사용자
- 언어/국가가 다른 사람과 가벼운 연결을 원하는 사용자

## Core Problem
- 기존 SNS는 빠르고 피상적이라 깊은 감정 교류가 어렵다.
- 글로벌 소통 앱은 실시간 채팅 중심이라 진입 피로도가 높다.
- “낯선 사람과의 연결”이 재미보다 불안으로 느껴진다.

## Positioning Statement
For 글로벌 감성 소통을 원하는 모바일 사용자, LetterGo는 지연 배송 기반의 편지 교환 경험을 제공하는 소셜 앱이다.  
기존 채팅/피드 앱과 달리, 여정이 보이는 지도형 전달 경험과 익명 기반 심리적 안전감을 함께 제공한다.

## Messaging Pillars
- 진심: 즉시 반응 대신 생각이 담긴 메시지
- 여정: 편지가 이동하는 시간이 감정적 기대를 만든다
- 세계성: 다른 나라의 사람과 부담 없이 연결된다
- 안전: 익명/거리 기반 설계로 노출 스트레스를 낮춘다

## Tone Guide
- 따뜻하고 간결하게 말한다
- 과장 대신 감정 장면을 보여준다
- 기능 설명보다 결과 경험을 먼저 말한다

## What To Avoid
- 데이팅 앱처럼 오해될 수 있는 문구
- “무제한/완벽” 같은 과대 표현
- 공격적 FOMO 문구만 반복하는 결제 유도

## Internal Tagline Candidates
- 마음은 천천히, 편지는 세계로
- 지구 반대편으로 보내는 한 줄의 진심
- 기다림이 설레는 글로벌 편지 앱

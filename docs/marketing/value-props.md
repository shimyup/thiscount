# LetterGo Value Propositions

## Segment-based Value

| Segment | Main Need | Value Message |
|---|---|---|
| Free 사용자 | 가볍게 시작하고 재미 확인 | 하루 3통으로도 글로벌 편지 경험을 시작할 수 있다 |
| Premium 사용자 | 더 자주, 더 풍부하게 소통 | 발송량 확대와 이미지/링크 첨부로 표현 폭이 커진다 |
| Brand/Creator | 국가 단위 도달과 브랜딩 | 대량 발송과 인증 배지로 글로벌 팬 커뮤니케이션 효율을 높인다 |

## Feature to Benefit Mapping

| Feature | User Benefit | Suggested Copy |
|---|---|---|
| 지도 배송 여정 | 기다림 자체가 콘텐츠가 됨 | “보내는 순간 끝이 아니라, 도착까지가 이야기” |
| 국가 랜덤 발송 | 탐험형 재미 | “오늘은 어느 나라로 닿을지 모르는 설렘” |
| 2km 근처 알림 | 현실감 있는 수신 경험 | “내 주변에 도착한 편지를 직접 확인” |
| 이미지+링크 첨부 | 감정 표현력 확장 | “문장에 장면을 더하세요” |
| 브랜드 대량 발송 | 운영 효율/도달 확대 | “한 번의 메시지로 여러 국가 팬에게 전달” |

## Plan Value Framing

### Free
- 핵심 경험 체험에 집중
- “먼저 써보고 맞으면 확장” 흐름 강조

### Premium
- 양(발송량) + 질(이미지/링크/특급) 동시 확장
- 권장 문구: “연결의 빈도와 표현의 깊이를 함께 늘리세요”

### Brand/Creator
- 팬 커뮤니티 운영 도구로 포지셔닝
- 권장 문구: “국가별 팬 접점을 반복 가능한 시스템으로”

## Objection Handling Copy

- “유료 전환이 너무 이른 것 같아요”
  - “먼저 무료로 충분히 경험하고, 필요할 때 확장할 수 있어요.”

- “하루 제한이 아쉬워요”
  - “프리미엄은 발송량 확대와 함께 이미지/링크 기능도 함께 제공해요.”

- “브랜드 플랜이 비싸 보입니다”
  - “국가 단위 다중 발송과 인증 배지를 포함한 운영형 플랜이에요.”

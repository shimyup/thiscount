# Letter Go vs Global Location-Based Games

Build 116 기준 글로벌 보물찾기·위치 기반 게임 경쟁 분석. Pokémon GO·Geocaching·
Ingress·Monster Hunter Now 가 이미 검증한 "걸어서 발견하는 재미" 메커니즘을
수입하되, **Letter Go 만의 불가모방 지점** 을 명확히 정의. 마케팅 기획서
Build 113 의 소비자 타깃 (20–35세 도심·쿠폰 탐색 익숙·게임화 리워드 친화) 에
맞춰 차별화 축을 정립.

## 🔑 Letter Go 만의 **불가모방 지점 5가지**

### 1. "콘텐츠 = 가치 = 즉시 사용"
Pokémon GO 에서 잡은 몬스터는 **소유물** (수집) 이다. 사용하려면 배틀이나
이벤트가 필요. Geocaching 의 트랙어블은 **여정 기록** 의 대상일 뿐 실사용
가치는 없음. **Letter Go 의 쿠폰 편지는 픽업 순간 = 사용 가능한 혜택**.
게임 진행 ≠ 가치 발생이 아니라, **게임 진행 = 즉시 가치 현금화**.

### 2. "게임 레이어 ⊃ 커머스 레이어"
Niantic 계열 앱들은 IAP (아이템·잼·Pro 멤버십) 으로만 수익화한다. **Letter
Go 는 브랜드 B2B 를 게임 루프 안에 내장** — 브랜드 ExactDrop 크레딧 ₩10,000 / 100통
은 게임 내 토큰이 아니라 실제 광고비. 유저·브랜드 양쪽 모두 실제 경제가치가
루프 안에서 순환.

### 3. "게임을 싫어해도 쓰게 되는 메커니즘"
포켓몬 고를 모르는 사람도 "할인 쿠폰" 은 안다. 게임 용어 (Gym·Pokéstop·CP·IV)
없이 누구나 1 분 안에 루프 이해. 20–35 세 타깃 중 게임에 거부감 있는 층을
커버 가능. **"포켓몬 고 스타일이지만 쿠폰" 이 아니라 "배민 쿠폰함인데 지도에서
줍는다" 프레이밍**.

### 4. "익명·비대결 — 심리적 안전"
Ingress 는 팀 PvP 의 긴장감, Pokémon GO 는 체육관 경쟁 스트레스. **Letter Go
는 혼자서도, 동네 친구와도, 관찰자로도 평등하게 경험 가능**. 지도에 나만 보이는
타워 + 익명 발신 편지 조합. 소셜 피로도 낮음 → 20 대 후반 ~ 30 대 중반 이후
은퇴 층에도 열림.

### 5. "서사 단위 = 편지"
Monster Hunter Now 의 수집은 장비·소재·수치. Pokémon GO 의 수집은 도감 숫자.
**Letter Go 의 수집은 "누군가 쓴 글"** — 브랜드의 프로모 카피·익명 회원의
여행 엽서. 같은 위치 기반 게임이지만 **문장 · 목소리 · 의도** 를 매 픽업마다
실제로 읽는다. 콘텐츠 포맷 자체가 타 앱에 없음.

## 🏁 Letter Go 고유 기능 (경쟁 앱에 없음, 지켜야 할 해자)

1. **카테고리 맞춤 이모지 일관성** (🎟·🎁·✉️ 가 compose → 마커 → 알림까지 동일) — Build 110·112
2. **3 단 개봉 애니 + 티어별 햅틱** — 포켓몬 GO 잡기 모션보다 내러티브 집중도 높음
3. **레벨업 = 줍기 반경 실제 확대** (+10m/Lv up to +490m) — 포켓몬 GO 는 레벨업 보상이 주로 cosmetic
4. **만료 타이머 내장 + 사이렌 배너** (Build 115) — 위치 기반 게임 중 유일하게 "시한부 보상"
5. **🎫 "사용 완료" 루프** — 픽업·오픈·사용 3 단계가 다 게임 안에서 감지됨. 브랜드 ROI 계산 가능 (Niantic 계열은 할인이 연동 안 됨)
6. **브랜드 팔로우·뮤트 양방향** (Build 115) — 유저가 선호를 능동 신호, 다른 위치 기반 게임에 이 개념 없음
7. **첫 픽업 축하 모먼트 + Hunt Wallet + Weekly Quest 3 단 온보딩** (Build 115·116)

## 🧭 후속 로드맵 (Build 117+)

**Phase 1 (즉시)**
- Brand Spotlight Week — 주간 특정 브랜드 이벤트 기획 (카피 + 일정)
- Brand Lure — 브랜드가 30 분간 주변 반경 ↑ 구매 옵션

**Phase 2 (2–3 개월)**
- 친구 초대 → 공동 주간 퀘스트 보너스
- 시간대/지역 큐레이션 룰 (아침 / 점심 / 저녁 / 주말)

**Phase 3 (6 개월+)**
- 타워 외관 커스터마이징 (브랜드 컬라보 스킨)
- 지역 히트맵 & 핫스팟 랭킹

# Letter Go Creative Kit

스토어 제출/홍보에 바로 쓰는 템플릿입니다.

## 포함 파일

- `appstore-67-template-1290x2796.svg`
- `appstore-65-template-1284x2778.svg`
- `play-feature-graphic-1024x500.svg`
- `social-post-1080x1080.svg`

## 사용 방법

1. SVG를 Figma/Canva/Illustrator에서 열어 앱 스크린샷으로 교체
2. 카피 문구를 출시 시점에 맞게 수정
3. 필요 시 `scripts/generate_marketing_assets.sh`로 PNG 내보내기

## 권장 문구

- 부제목: `세계로 보내는 랜덤 편지`
- 핵심 가치: `하루의 생각을 세계로 보내고, 누군가의 마음을 받으세요.`

## Long Description (Korean Draft)
Thiscount는 전 세계 어디선가 떨어진 할인 쿠폰을 지도에서 찾아 줍는 새로운 발견 앱입니다.

브랜드와 사용자가 매일 새로운 프로모션·이벤트 쿠폰을 세계 곳곳에 흩뿌립니다. 지도를 켜고, 가까운 쿠폰을 찾아 줍고, 매장에서 코드를 사용해보세요.

주요 기능

추천 대상

## Long Description (English Draft)
Thiscount is a map-based discovery app where brands and users scatter discount coupons across the world for nearby members to find and pick up.

Open the map, find a nearby coupon, pick it up within range, and use the code at the store. Every day, somewhere around you, someone is dropping a deal.

Key Features
- Location-based real-time coupon discovery
- Visual delivery routes on the world map
- Brand membership: send your own promotion messages
- Premium: pick up to 30 coupons/day, image+link attach, ad-free
- 14-language support

Best For
- Shoppers who love discovering new daily promotions and discounts
- Users who enjoy location-based gamification
- Brand operators who need a natural location-based marketing channel
